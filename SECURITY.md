# Security Policy

## Supported Versions

Only the latest version on the `main` branch is supported with security updates.

| Version | Supported |
| ------- | --------- |
| main    | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Option 1: GitHub Security Advisories**
Use the [Security Advisories](https://github.com/lewiswigmore/virus.xcheck/security/advisories/new) tab on this repository to privately report the issue.

**Option 2: Email**
Contact the maintainer at [lewiswigmore on GitHub](https://github.com/lewiswigmore).

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any potential impact

You can expect an initial response within 7 days. Please do not open a public issue for security vulnerabilities.
